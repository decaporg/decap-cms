# Безопасный CI/CD пайплайн для Decap CMS

## Общая структура
Пайплайн состоит из следующих этапов:
1. **SAST (CodeQL, Semgrep, ESLint)** — статический анализ кода.
2. **SCA (npm audit)** — проверка зависимостей.
3. **Secrets (Gitleaks)** — поиск секретов.
4. **DAST (OWASP ZAP)** — динамическое сканирование.
5. **Security Gateway** — контроль прохождения всех проверок.
6. **Build** — тест сборки перед деплоем.
7. **Security Summary** — сводка результатов и итоговая оценка.

## Автоматизация
Пайплайн реализован на GitHub Actions. Все файлы `.yml` расположены в `.github/workflows`.

## События
Проверки запускаются при:
- `push` в ветку main или feature;
- создании или обновлении `pull request`.

## Результаты
Все отчёты доступны:
- в разделе **Actions**;
- в **Security → Code Scanning Alerts**;
- в комментарии Security Summary.
