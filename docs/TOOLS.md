# Используемые инструменты

| Этап | Инструмент | Назначение | Зона роста |
|------|-------------|------------|-------------|
| SAST | CodeQL | Анализ кода на уязвимости | Добавить кастомные запросы |
| SAST | Semgrep | Анализ JS/TS на паттерны безопасности | Расширить правила |
| Lint | ESLint | Контроль качества кода | Включить TS strict |
| SCA | npm audit | Проверка зависимостей | Интеграция с Snyk |
| Secrets | Gitleaks | Поиск секретов | Проверка истории коммитов |
| Workflow | actionlint | Проверка корректности YAML | Pre-commit hook |
| DAST | OWASP ZAP | Динамическое тестирование | Активный режим |
| Gateway | GitHub Script | Проверка итогов | Уведомления в Telegram |
| CD | Deploy to VPS | Автоматическое развертывание | Добавить rollback |
