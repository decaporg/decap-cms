# Модель угроз Decap CMS

## Цель
Идентификация рисков безопасности и разработка мер защиты при построении DevSecOps пайплайна.

## Основные угрозы
| Тип | Пример | Контроль |
|-----|--------|-----------|
| Уязвимости в зависимостях | Пакеты npm с CVE | npm audit |
| Небезопасный JS-код | `eval`, `innerHTML` | Semgrep, ESLint |
| Утечка секретов | API keys в репо | Gitleaks |
| Ошибки конфигурации | .yml с неправильными шагами | actionlint |
| Веб-уязвимости | XSS, CSP ошибки | OWASP ZAP |

## Контроль и реакция
Все обнаруженные проблемы фиксируются в GitHub Security Alerts.  
Security Gateway блокирует merge при наличии критических уязвимостей.
